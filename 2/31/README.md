<h3>स्वधर्ममपि चावेक्ष्य न विकम्पितुमर्हसि |<br/>धर्म्याद्धि युद्धाच्छ्रेयोऽन्यत्क्षत्रियस्य न विद्यते ||२-३१||</h3>
