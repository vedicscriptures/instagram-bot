<h3>यत्तदग्रे विषमिव परिणामेऽमृतोपमम् |<br/>तत्सुखं सात्त्विकं प्रोक्तमात्मबुद्धिप्रसादजम् ||१८-३७||</h3>
<pre>yattadagre viṣamiva pariṇāme.amṛtopamam .<br/>tatsukhaṃ sāttvikaṃ proktamātmabuddhiprasādajam ||18-37||</pre>
