<h3>श्रीभगवानुवाच |<br/>काम्यानां कर्मणां न्यासं संन्यासं कवयो विदुः |<br/>सर्वकर्मफलत्यागं प्राहुस्त्यागं विचक्षणाः ||१८-२||</h3>
