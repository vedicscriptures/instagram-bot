<h3>लोभः प्रवृत्तिरारम्भः कर्मणामशमः स्पृहा |<br/>रजस्येतानि जायन्ते विवृद्धे भरतर्षभ ||१४-१२||</h3>
<pre>lobhaḥ pravṛttirārambhaḥ karmaṇāmaśamaḥ spṛhā .<br/>rajasyetāni jāyante vivṛddhe bharatarṣabha ||14-12||</pre>
