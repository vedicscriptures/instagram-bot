<h3>आदित्यानामहं विष्णुर्ज्योतिषां रविरंशुमान् |<br/>मरीचिर्मरुतामस्मि नक्षत्राणामहं शशी ||१०-२१||</h3>
<pre>ādityānāmahaṃ viṣṇurjyotiṣāṃ raviraṃśumān .<br/>marīcirmarutāmasmi nakṣatrāṇāmahaṃ śaśī ||10-21||</pre>
